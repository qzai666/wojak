# Wojak Draft Assistant

这是一个 Chrome 扩展 + 本地前端服务平台。平台负责提交 X 帖子链接并展示任务队列，扩展负责监听任务、打开 X、点赞、转发、评论附图，并把你自己评论生成的帖子链接回传到平台。

## 使用教程

### 1. 启动本地服务

打开终端，执行：

```bash
cd "/Users/qzai/Desktop/wojak 冲推/local-task-server"
node server.js
```

看到下面这一行表示启动成功：

```text
Wojak local task server listening on http://127.0.0.1:8787
```

停止服务时，在这个终端里按 `Ctrl + C`。

### 2. 打开前端页面

浏览器打开：

```text
http://127.0.0.1:8787
```

页面是暗黑模式。上方输入 `X 帖子链接`，按回车或点击 `加入队列`，可以连续加入多个链接排队。

下方任务队列会显示：

1. 目标链接
2. 已评论链接
3. 状态
4. 评论时间
5. 操作

评论完成后，`已评论链接` 后面会出现 `复制` 按钮，点击后会显示 `已复制`。

### 3. 安装 Chrome 扩展

1. 打开 Chrome。
2. 进入：

```text
chrome://extensions/
```

3. 打开右上角 `开发者模式`。
4. 点击 `加载已解压的扩展程序`。
5. 选择目录：

```text
/Users/qzai/Desktop/wojak 冲推/wojak-draft-extension
```

代码改动后，需要在 `chrome://extensions/` 里点击这个扩展的 `重新加载`。

### 4. 开始监听

点击 Chrome 扩展图标，弹窗里填写前端服务地址：

```text
http://127.0.0.1:8787
```

点击 `开始监听`。

开启后：

- 没有新链接时，扩展会打开 X 首页并自动滚动浏览。
- 首页随机点赞有自己的间隔，不影响三连任务。
- 有新链接时，扩展会按队列处理目标链接。
- 三连任务之间保持 2 到 10 分钟随机间隔。
- 面板只展示三连任务的操作间隔。

### 5. 自动三连流程

扩展监听到队列里的链接后，会自动：

1. 打开目标 X 帖子。
2. 点赞。
3. 转发。
4. 从根目录 `comments.json` 随机取一条评论文案。
5. 随机附加一张扩展素材图。
6. 发布评论。
7. 等待 10 秒。
8. 抓取你自己评论生成的帖子链接。
9. 回传到前端任务队列。

## 命令接口

也可以不用页面，直接用命令加入任务：

```bash
curl -X POST http://127.0.0.1:8787/api/wojak/tasks \
  -H 'Content-Type: application/json' \
  -d '{"targetUrl":"https://x.com/i/status/2053953866631160005"}'
```

查看任务和结果：

```bash
curl http://127.0.0.1:8787/api/wojak/tasks
```

## 注意事项

- 使用前请确认 Chrome 已登录 X。
- 本地服务必须保持运行，扩展才能监听任务。
- 如果页面或扩展代码改了，需要重启 `server.js` 或重新加载 Chrome 扩展。
- `comments.json` 位于项目根目录，扩展评论文案从这里随机读取。
